## step 0 - 준비하기

일단 시작하기 앞서 준비해둬야할것이 몇개가 있다.

> * Git
> * Git이 지원되는 IDE(통합 개발 환경)<br>    { ex - eclipse, Visual Studio, VS Code, Android Studio 등)
> * Git Hub

자 일단 가장 먼저 지금부터 사용할 Git을 설치해볼 것이다.
